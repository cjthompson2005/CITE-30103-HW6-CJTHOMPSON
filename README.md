# CITE-30103-HW6

This repository contains all the files for Homework 6 of CITE-30103.

## Files Included

- `question1.c` to `question25.c` — C programs for each question.
- `.txt` files — written answers for some questions.
- `screenshots/` — screenshots showing the output of each program.
- `README.md` — this file.

## How to Run Programs

1. Open a terminal.
2. Go to the folder where the files are located.
3. Compile a C program using: gcc questionname.c -o qnumber (q1, q25)
4. Run the program: using ./q1 ./q2 etc
5. Repeat for other programs as needed.

## Notes

- Each program has comments explaining what it does.
- Screenshots are included as proof that programs work.
